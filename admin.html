🔐 GitHub Connection

This panel saves directly to your GitHub repo via the API. You only need to enter these once — they're stored in your browser only, never sent anywhere except GitHub.


How to get a Personal Access Token

1. Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
2. Click Generate new token
3. Set repository access to your tracker repo
4. Under Permissions → Repository permissions → Contents: Read and write
5. Generate and copy the token above

The token lives in your browser's localStorage only. It is never embedded in any code.
